📊 Superstore Sales Dashboard

An interactive Sales Analytics Dashboard built using Python and Streamlit to explore and visualize Superstore sales data. The dashboard provides insights into sales performance, profits, regional trends, and category-wise analysis through dynamic charts and filters.

🚀 Features

✅ Interactive sidebar filters

✅ Region-wise and Year-wise analysis

✅ Sales and Profit insights

✅ Dynamic charts using Matplotlib and Plotly

✅ Date range filtering

✅ Refresh data option

✅ Clean and responsive UI


🛠️ Tech Stack
Python
Streamlit
Pandas
NumPy
Matplotlib
Plotly Express

Streamlit is a Python framework designed for quickly building interactive data applications and dashboards.
